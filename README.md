# bankcore360

# BankCore360 is a web-based Retail \& Corporate Banking Management Platform designed 

# for banks, credit unions, and financial institutions to unify customer accounts, transactions, 

# loan servicing, and compliance reporting. The platform consolidates customer data, account 

# balances, transaction history, and loan workflows to provide end-to-end visibility, streamlined 

# operations, and adherence to banking regulations.

# Roles:
# CSR, CO, CUSTOMER, LOAN_MANAGER, BRANCH_MANAGER, ADMIN
